# Module for Boxen

Install [CoronaSDK](http://www.coronasdk.com), a Lua-based, 2D game engine for Mac OS X.

## Usage

```puppet
include coronasdk
```

## Required Puppet Modules

* `boxen`

## Development

Write code. Run `script/cibuild` to test it. Check the `script`
directory for other useful tools.
